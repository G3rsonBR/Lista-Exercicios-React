Dominar a manipulação de arrays e a desestruturação (Destructuring) é o que dá "superpoderes" ao desenvolvedor JavaScript. No mundo da Orientação a Objetos, essas ferramentas são essenciais para processar coleções de objetos e extrair dados de forma limpa.

Nível Fácil (Sintaxe e Fundamentos)
Foco: Uso direto de map e desestruturação básica.
    1. Dobrar Valores: Use .map() para criar um novo array com o dobro de cada número de [1, 2, 3, 4, 5].
    2. Nomes Próprios: Dado um array de nomes em minúsculo, use .map() para retornar todos com a primeira letra maiúscula.
    3. Desestruturação Simples: Dado o array ['Maçã', 'Banana'], use desestruturação para armazenar cada fruta em variáveis chamadas f1 e f2.
    4. Troca de Variáveis: Use desestruturação de array para inverter os valores de duas variáveis a = 1 e b = 2 sem usar uma variável temporária.
    5. Objeto Pessoa: Dado {nome: 'Ana', idade: 25}, extraia o nome para uma variável usando desestruturação de objeto.
    6. Map com Parâmetros: Use o segundo parâmetro do .map() (o índice) para retornar um array de strings no formato: "Índice 0: Valor X".
    7. Conversão de String: Use .map() em um array de números para transformá-los em strings.
    8. Extrair Propriedade: Dado um array de objetos Cidades (com nome e populacao), use .map() para retornar apenas um array com os nomes das cidades.
    9. Desestruturação com Default: Tente desestruturar a propriedade cor de um objeto vazio, definindo um valor padrão "preto".
    10. Rest Operator Básico: Dado [10, 20, 30, 40], use desestruturação para pegar o primeiro elemento em primeiro e o resto em um array chamado outros.
